About This System

The ERVHS-EARIST Program Recommendation System is a rule-based intelligent system designed to help Senior High School students from Eulogio Rodriguez Vocational High School (ERVHS) make informed decisions about college programs at EARIST.

Unlike traditional personality tests, our system uses deterministic rules based on validated data from guidance counselors, curriculum analysis, and educational experts to provide transparent, defensible recommendations.

This system was developed as part of a Bachelor of Science in Computer Science thesis project at EARIST.

System Features

  • 30+ validated recommendation rules
  • 15+ EARIST program coverage
  • All 7 SHS strands supported
  • Confidence scoring for each recommendation
  • Detailed justifications provided
  • PDF report generation
